Projekt jest obsługiwany za pomocą Mavena

Istnieją paczki klienta i serwera gotowe do użycia.

Np. klient gotowy do uruchomienia jest w client/target
java -jar client/target/client-1.0-SNAPSHOT.jar nick

A server w server/target
java -jar server/target/server-1.0-SNAPSHOT.jar

Serwera nie da się wyłączyć "czysto" - polecam Ctrl+C

Hint:
Klawisz ENTER działa w kliencie tak jak się tego oczekuje.
